## Problem Definition:
Given a complete weighted graph and a starting node, what is the shortest tour

## Input:
Weighted adjacency lists in the following format:
n (number of nodes)
S (start node)
u v w (u: outgoing node, v: incoming node, w: weight)

Example:
```
3
A
A B 3
B C 2
C A 1
6
```
